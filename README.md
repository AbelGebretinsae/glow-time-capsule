# Time Capsule

## Full-Stack Time Capsule Web Application

A full-stack, character-based time capsule web application that allows users to create and store time-locked digital artifacts for future access.

### Features
- Create, store, and retrieve time-locked messages
- Character-based input system for structured entries
- Secure handling of locked content until release time

### Tech Stack
- **Frontend:** React
- **Backend:** Node.js
- **Database:** JSON-based persistent storage
- **API:** RESTful architecture

### Implementation Details
- Designed and implemented a RESTful API to support secure creation and retrieval of time-locked artifacts
- Built a custom React UI with dynamic routing and component-based architecture
- Managed application state and component lifecycle to ensure data consistency between client and server
- Applied modern web development best practices for maintainability and scalability

### Motivation
This project was built as a self-directed effort to strengthen full-stack development skills and explore time-based data access patterns.
